# Physical interaction within protein complexes links positive genetic interactions to disease vulnerability

## Overview

This repository contains all analysis code, figures, and the manuscript for our study investigating how protein complex architecture shapes genetic interactions and disease vulnerability in yeast.

We integrate genome-wide genetic interaction (GI), protein-protein interaction (PPI), protein complex, evolutionary, and disease data to analyze 42,979 gene pairs from the structurally resolved *Saccharomyces cerevisiae* interactome. We find that co-purified within-complex gene pairs preferentially exhibit positive genetic interactions (OR = 1.84), a pattern independently supported by evolutionary conservation and cross-species analyses. Disease gene orthologs, particularly cancer-associated genes, concentrate among these architecturally coupled pairs, supporting a "fragile complex" model linking physical proximity, positive epistasis, and disease vulnerability.

## Data sources

Raw data are from publicly available databases:

| Database | Description | Reference |
|----------|-------------|-----------|
| SGA | Genome-wide genetic interactions in yeast | Costanzo et al., *Science* 353, aaf1420 (2016) |
| Michaelis interactome | Structurally resolved yeast PPI network (31,004 interactions) | Michaelis et al., *Nature* 624, 192-200 (2023) |
| BioPlex 3.0 | Human PPI network (118,162 interactions) | Huttlin et al., *Cell* 184, 3022-3040 (2021) |
| HuRI | Human binary protein interactome (51,640 interactions) | Luck et al., *Nature* 580, 402-408 (2020) |
| CORUM | Mammalian protein complexes | Ruepp et al., *Nucleic Acids Res.* 38, D497-D501 (2010) |
| hu.MAP 2.0 | Human protein complex map | Drew et al., *Mol. Syst. Biol.* 17, e10016 (2021) |
| HPO | Human Phenotype Ontology | Kohler et al., *Nucleic Acids Res.* 49, D1207-D1217 (2021) |
| Evolutionary rates | Protein evolutionary rates in yeast | Xia et al., *PLoS Comput. Biol.* 5, e1000413 (2009) |
| Alliance of Genome Resources | Ortholog mapping | Alliance Consortium, *Genetics* 220, iyac022 (2022) |

## Key scripts

The analysis pipeline is organized as numbered scripts in `build/`:

- `01-08`: Initial data loading, overlap analysis, and early figures
- `10-30`: Deeper analysis, literature comparison, architecture classification
- `31-50`: Three-class architecture, master pair table, robustness tests
- `51-80`: Module analyses, evolutionary rates, cross-species conservation, disease analysis
- `81-99`: Figure generation, panel arrangement, manuscript writing
- `100-110`: Final figures (v48-v51), manuscript revisions (v43-v48), Science submission

The final submission package is generated by `build/110_science_submission.py`.

## Requirements
